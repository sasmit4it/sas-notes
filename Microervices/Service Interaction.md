Since all these will be independent services, we must figure out the best way for them to communicate with each other.

Some options are

synchronous messaging : client sends request and expects a response immediately i.e. blocks.

async messaging : Client sends request, but carries on with other tasks. It does not block for the response.

one to one messaging

one to many messaging
